Видеоредактор:
Работа с видео в формате mp4
Базовые операции с видео:
Изменение скорости
Вырезать фрагмент
Склеить фрагменты
Синхронизация операций с аудиодорожкой (т.е. если ускорили видео в 2 раза, то и аудио должно ускориться в 2 раза)
Поддержка истории изменений(в папке log)
Редактирование в терминах фрагментов
Расширенные возможности обработки :
Поворот картинки
Кроп

Склейка видео работает со всеми загруженными файлами

Брать файлы чтобы посмотреть работу, лучше низкого качества. Такие видео есть в папке "video".
